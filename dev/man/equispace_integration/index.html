AutoBZ.automatic_equispace_integration — Function
automatic_equispace_integration(f, a, b; kwargs)
automatic_equispace_integration(f, l::IntegrationLimits; npt1=0, pre1=0, npt2=0, pre2=0, pre_eval=generic_pre_eval, int_eval=generic_int_eval, atol=0.0, rtol=1e-3, npt_update=generic_npt_update, maxevals=typemax(Int64))

Automatically evaluates the integral of f over domain l to within the requested error tolerances atol and rtol. Allows optional precomputed data at two levels of grid refinement npt1, pre1 and npt2, pre2 as well as customizable precomputation with pre_eval and evaluation/summation int_eval. Moreover, a function defining an update strategy for npt can be passed as npt_update.

Precomputes the grid points and weights to use for equispace quadrature of f on the domain l while applying the relevant symmetries to l to reduce the number of evaluation points.
